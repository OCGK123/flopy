정교한 기능

flopy는 직관적인 인터페이스와 강력한 기능을 제공합니다

🔍

완벽한 투명도 지원

PNG 이미지의 투명 배경을 그리드로 시각화하여 쉽게 확인할 수 있습니다. 그리드 색상과 크기를 자유롭게 조절할 수 있어 작업 환경에 맞게 커스터마이징이 가능합니다.

✨

AI 배경 제거

내장된 자동 AI 배경제거 기술로 일반 이미지의 배경을 자동으로 제거하고 투명하게 만들 수 있습니다. 별도의 이미지 편집 도구 없이 한 번의 클릭으로 완벽한 결과물을 얻을 수 있습니다.

📌

고정 모드

이미지를 화면에 고정하고 클릭이 이미지를 투과하도록 설정할 수 있습니다. 트레이싱 작업이나 참조 이미지로 활용하거나 여러 이미지를 비교할 때 특히 유용합니다.

강력한 고급 기능

기본에 충실한 기능들을 가지고있습니다

01

투명도 조절

이미지의 투명도를 세밀하게 조절하여 작업 중인 프로그램과 같이 사용할 수 있습니다.

  • ✓ 실시간 투명도 조절
  • ✓ 다양한 배경 패턴 옵션
  • ✓ 작업 흐름에 자연스럽게 통합
02

크기 조절

이미지 크기를 쉽게 조절하여 다양한 목적에 맞게 활용할 수 있습니다.

  • ✓ 직관적인 슬라이더 컨트롤
  • ✓ 단축키를 통한 빠른 확대/축소
  • ✓ 화면에 맞춤/원본 크기 토글
03

항상 위에 표시

다른 앱이나 창을 사용할 때도 flopy를 항상 최상위에 표시할 수 있습니다.

  • ✓ 원클릭 활성화
  • ✓ 트레이싱 최적화
  • ✓ 디자인 작업 효율성 향상

자주 묻는 질문

flopy에 대한 궁금증을 해결해드립니다

배경 제거 기능은 어떻게 작동하나요?
배경 제거 기능은 rembg 라이브러리를 기반으로 한 머신러닝 기술을 사용합니다. 이미지의 주요 객체를 인식하고 배경을 자동으로 제거하여 투명한 PNG 이미지로 변환합니다. 정확도는 이미지 품질과 복잡도에 따라 달라질 수 있습니다.
flopy는 오픈 소스인가요?
아니요, flopy는 오픈소스가 아닙니다.
고정 모드는 무엇인가요?
고정 모드는 이미지 뷰어 창을 화면에 고정하고, 마우스 클릭이 이미지 영역을 투과하도록 하는 기능입니다. 이 모드에서는 이미지를 참조하면서 다른 창에서 작업할 수 있어 디자인 작업이나 이미지 비교에 유용합니다.
지원되는 이미지 형식은 무엇인가요?
PNG, JPG, JPEG, BMP, GIF 등 대부분의 일반적인 이미지 형식을 지원합니다. 특히 PNG 파일의 투명 배경을 시각화하는 데 최적화되어 있습니다.
macOS와 Linux 버전은 언제 출시되나요?
macOS와 Linux 버전은 현재 개발 중이며, 2025년 내에 출시될 예정입니다. 정확한 출시 일정이 확정되면 공식 웹사이트와 소셜 미디어 채널을 통해 안내드리겠습니다.
